Commands

This is a full list of commands that can be used in the command line while your server is running.

  • exit
    • An important command that stops both the command listening loop and the request handler.

Config

This list describes the values that can be changed in the app.config file.

  • esp_enabled
    • [true/false] Determines if .esp (Espresso Server Page) files are parsed. If false, they're just interpreted as text/html.
  • info_page
    • [true/false] Determines if the path /espresso_info returns info about the server, such as uptime and version.
  • engine
    • [engine name] Determines the scripting engine used to interpret server code in ESPs.
  • verbose
    • [true/false] Determines if ESP parsing is verbosely recorded in the console. Useful for internal debugging.
  • dir
    • [directory path] Determines the root directory for the server. Supports paths relative to the executable's directory.
  • port
    • [port number] Determines the port that the server runs on.
